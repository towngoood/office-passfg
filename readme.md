# Office 365 mailbox API APL

这是一个使用Office 365 API进行各种应用的程序。

## 安装

1. 克隆本仓库到本地。
2. 在终端中进入项目目录。
3. 运行`npm install`安装依赖项。

## 使用

1. 在终端中进入项目目录。
2. 运行`npm start`启动应用程序。
3. 打开浏览器并访问`http://localhost:3000`。

## API文档

请参考Office JavaScript API reference documentation。
